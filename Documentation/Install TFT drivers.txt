To install the required drivers and kernels to operate the capacitaive touch screen enter the following in a terminal

1) Remove unecessary packages and make sure the RPi is up to date:
	/// Removes unecessary packages
	$ sudo apt-get purge libreoffice*
	$ sudo apt-get purge wolfram*
	
	/// Updates packages to latest release
	$ sudo apt-get update && sudo apt-get upgrade
	$ sudo apt-get dist-upgrade
	$ sudo rpi-update

	/// Removes leftover dependencies and whatnot
	$ sudo apt-get autoclean && sudo apt-get autoremove

	/// Reboot for changes to take effect
	$ sudo reboot

2) Install TFT bootloader, drivers, and kernel:
	$ curl -SLs https://apt.adafruit.com/add-pin | sudo bash
	$ sudo apt-get install -y raspberrypi-bootloader adafruit-pitft-helper raspberrypi-kernel

3) Finally, configure the device overlays to allow the RPi to relay data to the TFT screen:
	$ sudo adafruit-pitft-helper -t 28c

	- You'll be prompted if you want the text console to appear on the TFT screen. Choose YES [y]
	- You'll be prompted if you want to map the GPIO pin #23 as an ON/OFF button. Choose YES [y]

4) Do a final reboot:
	$ sudo reboot

5) DONE!
